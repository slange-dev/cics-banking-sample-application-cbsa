#Sunset notice

The CICS Banking Sample application was released 3 years ago as a sample application, combining both classic CICS technology and modern Java and web browser based interfaces. It proved popular, with 61 forks, and has we believe been a useful contribution to the open source community.

There now exists a newer open source sample, "Bank of Z". https://github.com/IBM/Bank-of-Z

This combines all the best features of CBSA with additional IMS and MQ capabilities, with Zowe and DBB, and is the strategic sample going forwards. Because of this, the CICS Banking Sample Application will now be sunset, and will not be maintained or updated in the future.”

Thank you to everyone who downloaded, forked, expressed interest or installed CBSA!
